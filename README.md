# Внутренний учёт закупа и приёмки

Простое веб-приложение для ежедневной работы цехов **Суши / Паназия / Пицца**:
- выбор даты (по умолчанию сегодня по Казахстану);
- автоматическое создание листа дня по шаблонам;
- режимы **План** и **Приёмка**;
- автоматический расчёт расхождений и комментарии;
- закрытие дня вручную только закупщиком, либо автоматически в 00:00 по Казахстанскому времени;
- разграничение доступа по пользователям: старшие шефы видят только свой цех, закупщик — все цеха, админы (вы и владелец) — полный обзор;
- добавление новых товаров старшим шеф-поваром в лист следующего дня (закуп на завтра, до 00:00 KZ);
- закупщик не добавляет товары, а запускает кнопку **«Начать сверку»** для приёмки и контроля недопоставки/перепоставки;
- разделение товаров по категориям и быстрый поиск по наименованию;
- история с фильтрами и поиском;
- формирование списка перемещения для iiko по фактическим данным;
- адаптивный интерфейс для мобильных устройств (основной сценарий использования);
- гамбургер-меню с разделами: «Запланировать закуп», «Проверка закупа», «История», «Аккаунт» и кнопкой «Выход» внизу;
- интерфейс разделён по отдельным разделам меню, чтобы на одном экране не было всего сразу.
- в режиме сверки товары отображаются списком карточек для удобной поштучной проверки.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 4173
```
